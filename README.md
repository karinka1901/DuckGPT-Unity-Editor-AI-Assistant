# DuckGPT

An AI-powered rubber duck debugging assistant integrated directly into the Unity Editor.

https://github.com/user-attachments/assets/b5a05b76-b71f-42ee-98ab-60b29e3a4039

## Thesis Context

DuckGPT was developed as part of a Bachelor’s thesis in Game Development at Metropolia University of Applied Sciences.

The project explores how AI assistants can support debugging workflows within game development tools and examines the concept of rubber duck debugging in combination with modern large language models.

## Features

- AI-powered debugging assistant inside the Unity Editor
- Context-aware responses using Unity project data
- Automatic analysis of:
  - Console logs
  - Scene hierarchy
  - Selected scripts
- Conversation memory for follow-up questions
- Integration with the OpenAI API for AI-generated responses
- Optional text-to-speech responses via ElevenLabs
- Custom Unity Editor interface built with Unity UI Toolkit
